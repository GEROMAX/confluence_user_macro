# ユーザーマクロの作成方法
* 画面右上の歯車 > 一般設定 > ユーザーマクロ をクリック

* マクロ名(システム内)  
hogehoge_macro

* 表示設定 
「マクロブラウザーはシステム管理者のみ表示されます」を選択

* マクロ名(表示名)  
ほげほげツール

* カテゴリ  
「Confluenceコンテンツ」を選択

* ユーザーマクロの設定  
「マクロ本文はありません」を選択

* テンプレート  
任意のHTML＋JSなどを記載  
※最低限「## @noparams」の設定は必用
